The `cl-simplest-smtp` allows users to send emails using an SMTP server. It provides an easy-to-use interface for configuring the SMTP server settings and sending test emails.

To use the plugin, simply install and activate it on your WordPress site. Then, go to the plugin settings page and enter your SMTP server settings. Once you have saved your settings, you can use the contact form on your site to send emails using the SMTP server.
